PVH 簡化改名工具
==============

只區分兩大類：
  角度相 / 對色相

角度相：
  - 兩個非常相近的產品圖案 → AS
  - 單一產品正面 → FRONT
  - 側面 → SIDE
  - 角落 → CORNER

對色相：
  - 有色號 + 圖片上有 CWF 標籤 → CWF_色名/色號
  - 有色號但沒有 CWF 標籤 → D65_色名/色號
  - 色號例子：654-920 / 19-1555
  - 色名優先用 reference 色號表；OCR 讀到部份文字亦會模糊對照

可選：AI 睇圖改名（對色相 + 角度相）
  - Gemini 地區用唔到？改用：
      1) 通義千問 Qwen-VL（建議，香港／新加坡／內地可用）
         拎 key：阿里雲百煉／Model Studio
         base_url=https://dashscope-intl.aliyuncs.com/compatible-mode/v1
         model=qwen-vl-plus
         （內地改：https://dashscope.aliyuncs.com/compatible-mode/v1）
      2) OpenRouter：model=openrouter/free
      3) 本機 Ollama：model=llava（完全免費）
  - DeepSeek 官方 API 只得文字，睇唔到相
  - 複製 AI設定.example.txt → AI設定.txt，填 api_key，enabled=1
  - mode=always：每張都睇（對色 + AS/FRONT/SIDE/CORNER）
  - 環境變數：DASHSCOPE_API_KEY
  - 命令列：--ai / --no-ai / --ai-mode always|fallback|off

兩個模式：
  1_工作模式_自動改名.bat
      把 TDS + 未改名圖片放入「待改名圖片」後執行

  2_學習模式_訓練模型.bat
      把你改正確名嘅圖片放入「學習樣本」後執行
      學習完成後會清空「學習樣本」內嘅圖片/檔案（說明 txt 會保留）
      正確檔名例子：
        xxx_AS.jpg
        xxx_FRONT.jpg
        xxx_SIDE.jpg
        xxx_CORNER.jpg
        xxx_CWF_654-920.jpg
        xxx_D65_NAVY BLUE.jpg
