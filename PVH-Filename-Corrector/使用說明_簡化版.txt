PVH 簡化改名工具
==============

只區分兩大類：
  角度相 / 對色相

角度相：
  - 兩個非常相近的產品圖案 → AS
  - 單一產品正面 → FRONT
  - 側面 → SIDE
  - 角落 → CORNER

對色相：
  - 有色號 + 圖片上有 CWF 標籤 → CWF_色名/色號
  - 有色號但沒有 CWF 標籤 → D65_色名/色號
  - 色號例子：654-920 / 19-1555
  - 色名優先用 reference 色號表；OCR 讀到部份文字亦會模糊對照

可選：本機 Ollama AI 睇圖改名（對色相 + 角度相，免費）
  1) 安裝 https://ollama.com/
  2) 雙擊 0_準備Ollama_AI.bat（會 ollama pull llava）
  3) 保持 Ollama 開住
  4) AI設定.txt 已預設：
       enabled=1
       base_url=http://127.0.0.1:11434/v1
       model=llava
       mode=always
  5) 執行 1_工作模式_自動改名.bat
  - mode=always：每張都睇（對色 + AS/FRONT/SIDE/CORNER）
  - 可選更強：model=llama3.2-vision / moondream
  - 唔使雲端 API key；Gemini／DeepSeek 地區問題唔影響
  - 命令列：--ai / --no-ai / --ai-mode always|fallback|off

學習模式會學：
  - 角度相 vs 對色相
  - AS / FRONT / SIDE / CORNER
  - 對色相名記憶（你改正確嘅色名／色號）
      例子：xxx_D65_DESERT_SKY.jpg / xxx_CWF_WHITE.jpg
      工作模式會用色號對照 + 相似色卡影像幫手改名
  - 學習模式唔使開 Ollama


兩個模式：
  1_工作模式_自動改名.bat
      把 TDS + 未改名圖片放入「待改名圖片」後執行

  2_學習模式_訓練模型.bat
      把你改正確名嘅圖片放入「學習樣本」後執行
      學習完成後會清空「學習樣本」內嘅圖片/檔案（說明 txt 會保留）
      正確檔名例子：
        xxx_AS.jpg
        xxx_FRONT.jpg
        xxx_SIDE.jpg
        xxx_CORNER.jpg
        xxx_CWF_654-920.jpg
        xxx_D65_NAVY BLUE.jpg
